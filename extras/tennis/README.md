# Tennis Game Refactoring Kata (Java)
**Motivation:** Small, tangled logic into expressive, readable code; ideal for naming and flattening conditionals.

## How to Use
1) Bring a Java Tennis Game kata implementation into this folder.
2) Add parameterized tests for score progressions.
3) Simplify: replace nested ifs with clear state/lookup.

**What you learn**
- Guard clauses, table-driven logic, expressive naming
